# SSL Client

#### Description
Java SSL client

#### Usage
```
NAME
    ssl-client  -  Java SSL client

SYNOPSIS
    ssl-client COMMAND [OPTION]... 

OPTIONS
    Mandatory arguments to long options are mandatory for short options too.

    --help
        Show the help and exit
    -d, --debug
        Show debug message
    -v, --verbose
        Show verbose message
    -q, --quiet
        No output

COMMANDS
    help                Show the help text
    version             Show the version
    info                Show the Java info
    connect             Connect to a SSL server
    ciphers             Show the supported cipher suites
    test-ciphers        Test which ciphers work
AUTHOR
    Harold Li
```
